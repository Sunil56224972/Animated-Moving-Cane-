## 🚀 Overview

Welcome to my custom **3D Energy Drink Showcase**!

I developed this application to demonstrate interactive 3D product rendering on the web. It uses a custom-built algorithmic distortion shader to cycle through different brand textures directly on a 3D beverage can. To give it an energetic vibe, I integrated a reactive, wave-like background aura that emanates from the center and fills the viewport.

This serves as a boilerplate/demo for integrating advanced graphics programming into modern React applications.

## 🌟 Features

- **Custom Graphics Pipeline:** Bespoke fragment and vertex shaders for unique visual styling.
- **Interactive 3D Asset:** Real-time rendering of an energy drink container.
- **Fluid UI Animations:** Smooth state transitions seamlessly blending UI and 3D scenes.
- **Responsive Layout:** Automatically adapts the viewport and canvas sizing.

## 🛠️ Technology Setup

- **Frontend Core:** React 18, Vite
- **3D Engine:** Three.js, R3F (React Three Fiber)
- **Animation:** Custom Hooks, Framer Motion
- **Shading:** WebGL / GLSL

## 💻 Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Sunil56224972/Animated-Moving-Cane-.git
   cd Animated-Moving-Cane-
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
